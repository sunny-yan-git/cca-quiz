# CCA-F Exam Simulator

An interactive study tool for the **Claude Certified Architect – Foundations (CCA-F)** exam. Questions are dynamically generated by the Claude API and adapt to your chosen topic and difficulty.

## Features
- Dynamic question generation via `claude-sonnet-4-6`
- Topic and difficulty filtering (Easy / Medium / Hard)
- Per-session scoring with answer explanations
- Persistent score history in `data/scores.json`
- Seed question bank for offline fallback

## Prerequisites
- Python 3.11+
- Node.js 18+
- An [Anthropic API key](https://console.anthropic.com/)

## Setup

### 1. Clone & configure environment
```bash
git clone <repo-url>
cd cca-quiz

# Backend env
cp backend/.env.example backend/.env
# Edit backend/.env and set ANTHROPIC_API_KEY=sk-ant-...

# Frontend env (optional — defaults work with Vite proxy)
cp frontend/.env.example frontend/.env
```

### 2. Start the backend
```bash
cd backend
python -m venv .venv
source .venv/bin/activate  # Windows: .venv\Scripts\activate
pip install -r requirements.txt
uvicorn app.main:app --reload
```
API docs available at `http://localhost:8000/docs`

### 3. Start the frontend
```bash
cd frontend
npm install
npm run dev
```
App available at `http://localhost:5173`
